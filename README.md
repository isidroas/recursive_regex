# Recursive REGEX

Search an regular expresion recursively in a folder and substitute.

## Installation
```bash
pip install recursive_regex
```

### From source
```bash
git clone https://github.com/isidroas/recursive_regex
cd recursive_regex
# Install in editable mode
pip install -e .
```
